# menu_local_menu

## Requirements

### Developers

```shell
git hf
```

### Users

#### git

```shell
apt update
apt install git
```

#### myrepos

```shell
apt update
apt install myrepos
```

#### pandoc

```shell
apt update
apt install pandoc
```

## Installation (manual)

### manual installation

#### Ensure for path

```shell
PATH="$HOME/bin:$HOME/.local/bin:$PATH"
```

#### Ensure for relative link to menu script(s)

```shell
cd ~/bin
ln -sf menu/menu.sh menu-t

```
